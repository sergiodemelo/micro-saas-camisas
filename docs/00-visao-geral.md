# Visão Geral do Sistema – Micro SaaS de Pedidos e Vendas de Camisas

Este projeto tem como objetivo desenvolver um micro SaaS para gestão de vendas, pedidos e clientes de camisas de time, inicialmente para uso interno, mas com arquitetura preparada para expansão comercial (multi-loja, white-label e integração com IA).

## Objetivos Principais
- Registrar clientes, pedidos e itens de forma organizada.
- Calcular automaticamente custo, preço e lucro das camisas.
- Permitir anexar imagens para referência do fornecedor.
- Ser 100% responsivo (usável via celular).
- Preparado para futuro: IA, WhatsApp, catálogo integrado e afiliação.

## Público-Alvo
- Pequenos vendedores.
- Lojas de camisas.
- Revendedores independentes.

## Fases do Projeto
1. Banco de dados
2. Backend (APIs)
3. Frontend responsivo
4. Documentação e testes
5. Integrações futuras (IA/WhatsApp)
