# Endpoints da API

## Clientes
- POST /api/clientes
- GET /api/clientes
- GET /api/clientes/{id}

## Pedidos
- POST /api/pedidos
- GET /api/pedidos
- GET /api/pedidos/{id}

## Itens do pedido
- POST /api/pedidos/{id}/itens

## Uploads
- POST /api/pedidos/{id}/uploads

## IA (Futuro)
- POST /api/ia/criar-pedido
- POST /api/ia/criar-cliente
