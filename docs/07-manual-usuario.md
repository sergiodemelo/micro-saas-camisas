# Manual do Usuário

## Como cadastrar um cliente
1. Acesse o menu "Clientes".
2. Clique em "Novo Cliente".
3. Preencha nome e WhatsApp obrigatório.

## Como criar pedido
1. Vá em “Pedidos” → “Novo Pedido”.
2. Escolha o cliente.
3. Preencha itens.
4. Anexe imagens.
5. Salve.

## Como alterar dólar padrão
Configurações → Financeiro → Dólar padrão.
